# Hoskbrew Web v2

The official, high-fidelity retro-modern website for Hoskbrew.

## Tech Stack

- Next.js 16 (App Router)
- Tailwind CSS v4
- Framer Motion

## Features

- **Deep Retro Aesthetic**: CRT scanlines, neon accents, textured backgrounds.
- **3D Hero**: Floating mascot with parallax.
- **Interactive UI**: Beveled buttons, glitch text, cartridge-style cards.
- **RetroNomicon**: Digital zine interface.

## Setup

```bash
npm install
npm run dev
```

## Asset Usage

- **Mascot**: `Color_Octopus_New.png`
- **Logo**: `W_B_Octopus_New.png`
- **Watermark**: `B_W_Octopus_New.png`
